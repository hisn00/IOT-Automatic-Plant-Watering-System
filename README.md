##  **How It Works**  
1. **Monitor Water Levels**:  
   - Ultrasonic sensor captures real time water levels in the tank.  
   - Data is displayed dynamically on the GUI.  

2. **Automated Control**:  
   - Motor turns ON/OFF based on predefined water level thresholds.  
   - Manual control option included in the GUI.  

---

## 🚀 Future Improvements

- ☔ **Rain Detection:** Pause watering during rain using a sensor or weather API to save water.
- 🌿 **Fertilizer Dispenser:** Add a second pump to auto distribute liquid fertilizer on schedule.
- ☀️ **Solar Powered:** Use solar panels to run the system sustainably.
